API Tiedot

Palaa takaisin kotisivulle

Tämä avoin rajapinta tarjoaa tietoa Helsingin vegaaniravintoloista:

Käyttöliittymässä voit lisätä, poistaa, etsiä ja muokata

Alla olevia API-kutsuja voi vapaasti käyttää. Tiedot tulevat JSON-muodossa

Tyyppi Kutsu Tiedot Esimerkki
GET /restaurants Hakee kaikki ravintolat
GET /api/restaurants/search/:searchBy/:param Hae ravintoloita valitsemasi parametrin perusteella /api/restaurants/search/city/Helsinki Palauttaa Helsingissä olevat ravintolat
GET /restaurant/:id Hakee ravintolaa id:n perusteella /restaurant/1 Palauttaa ravintolan, jonka id on 1
